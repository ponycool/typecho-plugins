## Typecho 插件
---
在typecho的使用过程，很多需求没有找到合适的插件，又不想直接修改核心代码影响升级，所以造了一些轮子。

### 版本说明

插件版本需要和Typecho版本保持一致

#### 当前版本 V1.2

环境要求
- PHP 8.0 或更高版本
- Typecho v1.2.x

#### V1.0

- PHP 5.6 ～ PHP 7.4
- Typecho v1.1
