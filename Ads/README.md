# 广告投放

### 使用方法

1. 下载本插件，放在 usr/plugins/ 目录中
2. 登录管理后台，激活插件
3. 后台设置添加广告位
4. 在需要显示的位置，添加显示代码

```php
# 显示默认的广告代码位
# 语法: Ads_Plugin::show();

# 显示指定ID的广告代码位
# 语法: Ads_Plugin::show(id:1);

# 显示指定广告位名称的广告代码位
# 语法: Ads_Plugin::show(adName: '广告位名称1');

# 显示代码要用放在PHP代码块中，例：
<?php Ads_Plugin::show(); ?>
```
