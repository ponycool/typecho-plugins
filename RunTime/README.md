## Typecho 运行时间插件 RunTime v1.2.0
---
显示站点运行时间

**使用方法**

1. 下载本插件，放在 usr/plugins/ 目录中
2. 登录管理后台，激活插件
3. 在需要显示的地方调用显示方法

```php
语法: RunTime_Plugin::show();
```
